# 要件定義書 - Food Order

## プロジェクト概要
飲食店向けモバイルオーダーアプリ（LINEミニアプリ）

## 機能一覧

### 1. メニュー表示・注文
- メニュー一覧表示（カテゴリ別）
- 商品詳細表示
- カート機能（追加・削除・数量変更）
- 注文確定

### 2. 注文履歴
- 過去の注文一覧
- 注文詳細表示
- ステータス確認（注文受付・調理中・提供済み）

### 3. 店員呼び出し
- ワンタップで店員呼び出し
- 呼び出し理由選択（おかわり・お会計・その他）

### 4. 精算依頼
- 会計依頼ボタン
- 注文合計表示
- 支払いはレジで行う

### 5. LINE通知
- 新規注文をLINE公式アカウントに通知
- 店員呼び出しをLINE公式アカウントに通知
- 精算依頼をLINE公式アカウントに通知
